# UltimateFriends (Velocity Port)

Questo plugin è un porting per Velocity del plugin originale **UltimateFriends**, creato inizialmente per BungeeCord dallo sviluppatore **M1nef4n**.

## Dettagli del Porting
Durante lo sviluppo abbiamo semplicemente tradotto tutti i metodi e le API di BungeeCord nei rispettivi equivalenti per **Velocity**, mantenendo il funzionamento interno e la logica del plugin completamente identici all'originale.

## Destinazione d'uso
Questo port è stato realizzato specificatamente per **TrolladiaNetwork**.
